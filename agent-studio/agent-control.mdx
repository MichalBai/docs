## Agent Org Chart

Manage all your agents in a visual org chart, see connections, roles, and tasks across your entire team. Enable, disable, or add agents at any time. Run your AI team like a company: clear structure, full visibility, no complexity.

Your orchestrator — named **Jake** by default, but you can rename it to anything — runs the operation. It briefs every sub-agent, delegates work, and quality-checks all outputs. New goal? Add an agent. Your team grows in seconds, not quarters. Zero management overhead.

## Agent Task Board

Each agent has its own task board showing current work, priorities, and progress in real-time. Add, pause, or remove tasks at any time, full transparency into what every agent is doing without having to ask.

For every task you can see: status (backlog, in progress, paused, recurring), cost, model used, output produced, last run time, and next scheduled run. Nothing happens in the dark.

Tasks are organized by status:

| Status | Meaning |
|---|---|
| **Backlog** | Queued, not started yet |
| **In progress** | Currently being worked on |
| **Paused** | Manually paused |
| **Recurring** | On a schedule, runs automatically |
| **Completed** | Finished successfully |
| **Failed** | Encountered an error |

Click into any task to see its subtasks, the prompt it used, and a progress log. To add a task manually, click **Add Task** or send the instruction directly via your connected channel.

## Background task scheduling

Set up tasks, routines, and workflows with deadlines and priorities. Agents execute them in the background on their own schedule, work gets done while you sleep. Truly autonomous execution.

## Custom Approval Workflows

Define exactly how much autonomy your agents have:

- Set which tasks require your approval before execution
- Approve or reject agent actions from any channel (in-app, Slack, email)
- Adjust boundaries at any time

Start with more control and scale autonomy as your confidence grows. There's no binary choice between full manual and full handoff.

## Spending caps

Set a monthly spending limit per workspace to control how much each agent can spend on AI compute. When a workspace hits its cap, the agent pauses and notifies you. Useful for agencies managing multiple client workspaces, each client's costs stay contained and predictable.

You can also verify exactly what each agent spent, which model it used, and what output it produced, full cost transparency at the task level.

## Activity log

Go to **Settings > Activity** to see a full history of what your agents did, which tasks ran, how many credits each task consumed, and what output was produced. Use this to audit spend, verify output quality, and understand exactly where your credits went.
