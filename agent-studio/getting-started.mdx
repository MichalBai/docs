Agent Studio is where you create and manage your AI agent team. Agents operate autonomously — executing tasks, managing your content library, monitoring performance, and publishing content without you initiating every action. Build them by describing what you want. No coding or node-building required.

## The Agent Studio dashboard

When you open Agent Studio, the overview tab shows:

- **Total runs** — how many agent tasks have run this month
- **Tasks completed** — completed tasks for the period
- **Pending approvals** — items waiting for your review
- **Activity frequency** — a graph showing when your agents are most active throughout the day and week

<Tip>
  Ideally, you want to see activity across your week. If you don't see any activity, it usually means something isn't set up or connected yet.
</Tip>

## Your default agent team

Your agent team is ready to use on day one. It ships with built-in research tools, winning workflows, and proven skills — no configuration required. Start getting results immediately, then customize as you go.

Your main agent — the orchestrator — is named **Jake** by default, but you can rename it to anything you want. Jake coordinates all the sub-agents and is the one you communicate with directly.

## Building agents via chat

Create, train, and deploy agents by chatting with them. Describe what you want an agent to do, and Pressmaster builds it. Adjust its behavior, give it new skills, and deploy — all through conversation.

## Playbook Library

Access a library of pre-built agent systems and skills — including GTM Sales Agents, AI GEO Optimizers, and more. Deploy complete, specialized agent setups with one click, then iterate from there. No need to build from scratch.

## Custom Skill Library

Upload, create, and manage custom skills for your agents. Build new skills by chatting with the agent about what you need. Train agents to your exact way of working — no limits on what they can do.

## Communicating with your agents

Chat with your agents from wherever you are:

- **In-app** — directly within Pressmaster
- **Slack** — send instructions and get updates in your workspace
- **WhatsApp / Telegram** — delegate tasks from your phone
- **Email** — loop agents into your existing workflows

Context is maintained across all channels. Delegate from anywhere — text your agent team like you would a colleague.

**Which channel should you choose?** Pick the app you already use most — the goal is to remove friction, not add it. If you live in Slack, connect Slack. If Telegram or WhatsApp is your main messaging app, use that. The more natural the channel feels, the more consistently you'll use it.

## What agents can do for you

- Create drafts and manage your Content Library
- Review analytics and surface performance insights
- Publish content to connected channels
- Monitor industry trends and competitor activity
- Execute research tasks and store findings in the Artifact Library
- Run background tasks on a schedule — without you logging in

## Recommended usage pattern

You don't need to live inside Pressmaster for it to work. A common and effective pattern:

- **On your phone** — use your connected messaging app (Telegram, WhatsApp, or Slack) to send quick ideas, save thoughts, and check in with your agent throughout the day
- **In the web app** (~twice a week) — review tasks, approve content, check what the agent has been doing, and run interviews

The agent handles the work in between. You just guide and approve.

<Tip>
  Start with the default agent team before building custom agents. The pre-built setup covers most content workflows and gives you a solid baseline to iterate from.
</Tip>
