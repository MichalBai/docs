## Overview

Agents Managing Agents is an agency-focused feature that introduces a hierarchy of agents, a top-level agency agent that manages, communicates with, and coordinates sub-agents across multiple client workspaces.

### How it will work

**Agency-level agent**, your agency has a single master agent that oversees all client workspaces and their individual agents.

**Sub-agents per client**, each client workspace has its own agent that handles content creation, publishing, and reporting for that client.

**Agent-to-agent communication**, your agency agent can query sub-agents, assign tasks, collect reports, and coordinate activity across all clients without you manually checking each workspace.

**Example:**
> Send your agency agent: *"Give me a report on James, what's working, what isn't, and what should we change?"*
> 
> The agency agent queries James's sub-agent, collects the data, and delivers a consolidated report.

### What you'll be able to do

- Request performance reports across all clients from one place
- Assign tasks to specific client agents via your agency agent
- Set cost controls and spending limits per client workspace
- Create and deploy your own skill library and agent library across all clients, your reusable blueprint that you import for every new client

### Why it matters for agencies

Your skill library and agent library become your core product. Instead of selling time, you're deploying a tested, optimized system, pre-built agents, proven workflows, and fine-tuned skills, that you apply to every client from day one.
